# text_to_speech — Speech synthesis

Use `text_to_speech` when the user wants to hear a message or needs an audio file they can play back.

- Write the text the way it should sound out loud — expand abbreviations, spell out numbers when tone matters, and use punctuation for pacing.
- Let defaults pick the voice and provider unless the user has asked for a specific one. Don't switch providers mid-conversation without a reason.
- The generated audio is saved to the workspace; share the path so the user can find it.
