# web_search — Search the web

Use `web_search` when the answer depends on current information, recent events, or sources outside your training.

- Phrase queries the way you'd type them into a search engine — short keyword phrases, not full questions.
- Run narrow follow-up searches when the first page is ambiguous, instead of asking the user to refine what the web says.
- Summarize findings in your own words and cite the source URL for any specific claim, number, or quote.
