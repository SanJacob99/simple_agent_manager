# music_generate — Music and ambient audio

Use `music_generate` when the user asks for original music, ambient loops, or a musical sketch.

- Describe genre, mood, tempo, instrumentation, and approximate length. "Gentle lo-fi piano, 60 bpm, 30 seconds" beats "chill music".
- Generation takes several seconds. Wait for the result rather than retrying impatiently.
- The file is saved to the workspace; share the path so the user can play it back.
